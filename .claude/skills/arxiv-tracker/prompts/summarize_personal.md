## 输出要求

生成结构化 Markdown 报告，格式如下：

1. 开头说明：「以下论文不直接关联业务方向，但在研究方法、技术趋势等方面有参考价值。」
2. 所有论文按 overall 评分降序排列
3. 每篇论文格式：

```
⭐ X.X | 英文标题
[标签1] [标签2]
• 作者: 作者列表（前3位 + et al.） | 机构信息
• 亮点: 核心技术贡献（1句话）
• 关注点: 值得关注的具体方法或思路

```

4. 作者行：作者列表后用 ` | ` 分隔附上 affiliations 字段（如有）
5. 标签使用论文数据中的 tags 字段
6. 语言：中文为主，专业术语保留英文
7. 不要添加开头的寒暄或结尾的总结，直接输出报告内容
